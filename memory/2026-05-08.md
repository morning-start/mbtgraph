### 关键发现

1. **`derive(Show)` 已废弃**：MoonBit 最新版本中 `derive(Show)` 已被标记为废弃，建议使用 `derive(Debug)` 或手动实现 `Show` trait
2. **`pub using` 用于跨包重新导出**：`pub using @pkg { ... }` 语法用于重新导出**其他包**的 API，同一包内无需额外导出
3. **`pub(all)` vs `pub`**：`pub(all)` 限制模块内可见，`pub` 完全公开（库 API 应使用 `pub`）
4. **trait 方法签名简化**：`fn method(self : Self, ...)` 可简化为 `fn method(Self, ...)`

## 操作：修复 adjacency_matrix.mbt 邻接矩阵实现

### 修复过程

经过多轮迭代修复，解决了大量 MoonBit 语法兼容性问题：

| 问题 | 旧语法 | 修复方案 |
|------|--------|----------|
| 可变局部变量 | `let mut` / `var` | 均不支持 → 改用尾递归 |
| 不等于比较 | `!==` / `===` | 不存在 → 改用 `match` 模式匹配 |
| 枚举构造 | `GraphError.NodeNotFound(...)` | 改为 `GraphError::NodeNotFound(...)` |
| 多行 impl with | `impl T for Type with { fn a... fn b... }` | 每行独立 `impl T for Type with a(...) {...}` |
| with 块内局部函数 | `fn helper(...)` 在 with 块内 | 提取为模块级函数 |
| Unit 值 | `Ok(Unit)` | 改为 `Ok(())` |
| 矩阵初始化共享引用 | `Array::make(n, row)` 全部共享同一行 | 逐行初始化 `matrix[i] = Array::make(capacity, None)` |

### 辅助函数提取

提取了 6 个模块级辅助函数：
- `has_node_at` / `has_edge_at` — 检查节点/边是否存在

- `count_row` / `count_row_from` — 计数行边数（递归）
- `count_col` / `count_col_from` — 计数列边数（递归）
