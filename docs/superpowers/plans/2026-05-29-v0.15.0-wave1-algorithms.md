# v0.15.0 Wave 1 — Classic Algorithm Expansion

> **For agentic workers**: Implementation plan for TASK-1A/1B/1C.

**Goal**: Add link prediction suite, dense subgraph primitives, and community detection enhancements to mbtgraph.

**Architecture**: Each algorithm group is a new module under `lib/algo/`, following existing patterns (trait-compatible `[G : @core.GraphReadable]`, `pub(all)` result structs, `derive(Debug)`).

**Tech Stack**: MoonBit, GraphReadable trait, DirectedAdjList/UndirectedAdjList storage.
