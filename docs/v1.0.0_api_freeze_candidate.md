## 📌 冻结政策

### 什么是 API 冻结？

从 v1.0.0 开始，以下公共 API 将**保持向后兼容**：
- ✅ 已有函数签名不会改变
- ✅ 已有类型字段不会删除或重命名
- ✅ Trait 约束只会放宽（不会收紧）
- ⚠️ 新增 API 可能添加（Minor 版本）
- 🚫 废弃 API 将在 2 个 Major 版本后移除

### 兼容性保证

| 版本范围 | 保证内容 |
|---------|---------|
| v1.x.x | 所有 v1.0.0 冻结的 API 完全兼容 |
| v2.0.0 | 仅移除 v1.0.0 标记为 `@deprecated` 的 API |
| v3.0.0+ | 可能引入 Breaking Changes（需 Migration Guide）|

## ✅ 存储层 API (storage/) — 🔒 冻结

### 8 种存储类型

| 存储类型 | Trait 支持 | 适用场景 | 复杂度 |
|----------|:----------:|---------|--------|
| `DirectedAdjList` ⭐ | R+W+D+E | 通用有向图 | 邻居 O(k) |
| `UndirectedAdjList` ⭐ | R+W+E | 通用无向图 | 邻居 O(k) |
| `DirectedMatrix` | R+W+D | 小型稠密图 (<1K) | O(V²) 空间 |
| `UndirectedMatrix` | R+W+E | 小型无向稠密图 | O(V²) 空间 |
| `EdgeList` | R+W+E | MST/Kruskal 友好 | O(E log E) |
| `UndirectedEdgeList` | R+W+E | 无向边集 | O(E log E) |
| `CSRGraph` | R+B | 大规模静态图 (>100K) | 缓存友好 |
| `CSCGraph` | R+B | 入边密集查询 | in_degree O(1) |

### 工厂函数

```moonbit
// 推荐：使用工厂函数创建实例
let g = @storage.new_directed()           // DirectedAdjList
let ug = @storage.new_undirected()         // UndirectedAdjList
let el = @storage.new_edge_list()          // EdgeListGraph
let m = @storage.new_directed_matrix(100)  // DirectedMatrix (预分配容量)

// Builder 模式（CSR/CSC 专用）
let csr = CSRBuilder::new()
let csr = CSRBuilder::add_node(csr, 0.0)
let csr = CSRBuilder::add_edge(csr, 0, 1, 3.14)
let graph = CSRBuilder::build(csr)
```

### 批量操作 (v0.14.0+)

```moonbit
// 跳过重复检查的高性能添加（调用方保证不重复）
let g = DirectedAdjList::add_edge_unchecked(g, from, to, weight)

// 批量添加（任一失败则终止）
let g = DirectedAdjList::add_edges_batch(g, edges)
```
