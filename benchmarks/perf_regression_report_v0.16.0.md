## 变化分析

### 一致性的微小升高 (+5~10%)

所有算法均存在 5-10% 的微小耗时增加，且**分布均匀**（不特定于某个算法）。这表明根因是**外部环境变化**而非代码回归：

1. **MoonBit 编译器版本更新**（2026-05-29 → 2026-06-09 间可能有更新）
2. **core 库版本变更**（可能导致间接的微小性能变化）
3. **WASM 运行时环境差异**

### 关键观察

| 算法 | 观察 | 判断 |
|:----|------|:----:|
| **CSR 转换** | 各规模仅 +2~3.6%，最稳定 | ✅ v0.14.0 优化持续有效 |
| **Dinic** | 各规模 +3~10%，标准差 3~4% 较高 | ✅ 无异常，噪声正常 |
| **Dijkstra** | 大规模 +13.8% 略高，但 SD 仅 1.2% | 🔍 建议下次复查 |
| **Louvain** | 仅 +6.3%，SD 仅 0.98% | ✅ v0.14.0 优化持续有效 |
